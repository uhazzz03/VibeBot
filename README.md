# 🎧 VibeBot 🤖
A chatbot that recommends songs and playlists on Spotify based on your mood, activity and personality preferences and will be developed further to consider artist and album searches as well and include a better ML model.

## 🌏 Live Demo
🔗 https://vibebotformusic.streamlit.app/

## 🚀 Features

* Chat-based interface 
* NLP-based Machine learning mood classification
* Personalized recommendations
* User preferences and Activity detection
* Track and playlist recommendations
* Saved user profile system
* Developer mode added

## 🔧 Tech Stack

### 💻 Frontend
- Streamlit

### ⚙️ Machine Learning (NLP model)
- Scikit-learn (TF-IDF and added classification pipeline)

### 📊 Data Processing
- Pandas

### 🛠️ Deployment
- Streamlit community cloud

### 🌏 APIs
- Spotify WEB API

## 🛠️ Local Setup
- git clone https://github.com/your-username/vibebot.git
- cd vibebot

- python3 -m venv venv
- source venv/bin/activate

- pip install -r requirements.txt
- streamlit run app/main.py

## 🔑 Environment Variables
- SPOTIPY_CLIENT_ID=your_client_id
- SPOTIPY_CLIENT_SECRET=your_client_secret
